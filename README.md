   ```sh
   npm install

3. Set up environment variables in a `.env` file:
   ```env
   PORT=3000
   DATABASE_URL=your_database_url
   JWT_SECRET=your_secret_key
   ```
4. Run the server:
   ```sh
   npm start
   ```
5. The API will be available at `http://localhost:3000`.
